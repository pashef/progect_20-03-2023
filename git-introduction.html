git log покажет нам все комиты проекта

  • Чтобы не вносить в комит изменения файла поможет команда: git cheсkout имя файла

Если мы сделали изменения в нескольких файлаи и добавили их в комит git add . и если смотреть git status то 2 файла теперь в слежке и если мы не хотим их комитить вместа, то можно удалить не нужные из индексного файла

  • git reset HEAD путь до нужного файла
  • git show хеш комита - посмотрим какие изменения произошли в файлах
  • git commit --amend -m " Новый комент " Позволяет сменить нам неверно написаный коментарий в коментах
  • git commit am хеш комита - посмотрим какие изменения произошли в файлах
  • git log -1 покажет последний комит
  • git rm путь к файлу - Команда позволяет удалить не нужный файл из комита
  • git commit --amend --no-edit - Делаем комит без изменения сообщения последнего комита - это позволяет команда --no-edit
  • Если мы хотим удалить файл с комита,но не хотим удалять его с тела индексного дока - git rm --caсhed путь к файлу, далее делаем комит git commit --amend --no-edit, Проверим изменения в удаленом файле и видим что изменеия в нем удалились а сам файл вернулся в комит.
